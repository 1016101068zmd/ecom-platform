# ecom-platform
An e-commerce platform supporting user login, product browsing, cart, checkout, and admin management, built by a three-person team.
一个三人协作的电子商务平台课程项目，包含用户端商城与后台管理端，覆盖电商基础流程：商品浏览、购物车、下单与订单管理、后台商品与订单维护。

主要功能
用户端（前台）
用户注册/登录
商品列表/商品详情
搜索/筛选（可简化为按分类或关键词）
购物车（加入、修改数量、删除、查看）
下单/结算（可模拟支付）
我的订单（列表/详情/状态展示）
管理端（后台 Admin）
管理员登录（可与普通用户做简单角色区分）
商品管理：新增/编辑/上下架/删除、库存与价格调整
订单管理：查看订单、修改订单状态（如：待支付/已支付/已发货/已完成/已取消）


项目分工
三人代码工作量尽量均衡，模块边界清晰，可通过 Git 提交记录证明贡献。

成员 A：后端核心（backend）
数据库表设计与数据模型
用户模块：注册/登录
商品模块：商品列表/详情/搜索筛选接口
购物车模块：增删改查接口
订单模块：创建订单、订单列表/详情、状态流转
输出接口文档：docs/api.md

成员 B：用户端前台（frontend）
登录/注册页、商品列表/详情页、购物车页、结算下单页、我的订单页
公共组件（导航栏、商品卡片、分页/加载、提示）
按接口文档对接后端 API
页面表单校验与基础异常提示
首先完成登录注册界面并且预留好跳转接口
其次完成商品列表的导航栏和主界面的网页

成员 C：后台管理端（admin-panel）
管理员登录与权限区分
商品管理（表格、编辑弹窗/页面、上下架等）
订单管理（列表、详情、修改状态）
（选做）用户管理
后台界面基础能力：搜索/筛选、分页、操作确认弹窗
